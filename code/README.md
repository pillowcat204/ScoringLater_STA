# Code folder

This folder contains scripts used for data processing, analysis, and results table generation for one simulation condition.

## Contents

- `cat_2.R`: simulation script for one representative condition
- `calc_pairwise_overlaps.R`:helper function for pairwise overlap calculations

## Expected workflow
1. Clone or download this repository.  
2. Open R in the project directory.  
3. Install required packages if needed.  
4. Run:

```r
source("code/cat_2.R")
```

The script will generate an output file in the `output/` folder.
